# 🔤 Spelling & Grammar Corrector

A Python-based NLP application that detects and corrects spelling and grammar errors using **Edit Distance**, a **Noisy Channel Model**, and a **Bigram Language Model**. The project features an interactive Tkinter GUI with real-time suggestions and automatic text correction.

## ✨ Features

- Spelling error detection
- Grammar rule checking
- Edit Distance (Levenshtein)
- Noisy Channel Model
- Bigram Language Model
- Auto-correction
- Interactive Tkinter GUI
- Context-aware suggestions

## 🛠️ Technologies

- Python
- Tkinter
- NLTK
- pyspellchecker

## 🚀 Installation

```bash
git clone https://github.com/your-username/Spelling-Corrector.git
cd Spelling-Corrector
pip install nltk pyspellchecker
python main.py
```
